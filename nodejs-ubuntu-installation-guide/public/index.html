Installing nodejs and npm with Node Source

  1. Remove the existing npm and Node.js installations:

    sudo apt remove npm && sudo apt remove nodejs

  2. Update the package lists:

    sudo apt update

  3. Install Node.js and npm using the NodeSource PPA (Personal Package Archive). Run the following commands to add the NodeSource repository and install Node.js:

    curl -sL https://deb.nodesource.com/setup_lts.x | sudo -E bash - sudo apt install -y nodejs

  4. Verify that Node.js and npm are installed correctly by running the following commands:

    node --version npm --version
    5. These commands should display the installed versions of Node.js and npm, respectively.

Installing nodejs and npm with Node Version Manager (NVM):

  1. Remove the existing npm and Node.js installations:

    sudo apt remove npm sudo apt remove nodejs

  2. Install Node Version Manager (NVM), which allows you to manage multiple versions of Node.js:

    curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash source ~/.bashrc

  3. Verify that NVM is installed correctly by running the following command:

    nvm --version

  4. Install a stable LTS version of Node.js using NVM:

    nvm install --lts

  5. Verify that Node.js is installed correctly by running the following command:

    node --version

  6. Update npm to the latest version:

    npm install -g npm@latest

  7. Verify that npm is installed correctly by running the following command:

    npm --version
    8. With these steps, you have successfully reinstalled npm in Ubuntu. Please note that using NVM allows you to switch between different Node.js versions easily. You can install and use different versions by running nvm install <version> and nvm use <version>, respectively.

    It's important to keep your Node.js and npm versions up to date for security and compatibility reasons.
